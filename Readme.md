## Installation

- Do npm install on both folder i.e. client & server
- Setup MongoDB and edit server/.env file

## Usage

- After installation

- TO Run Client

```bash
npm run dev
http://localhost:8000/
```

- TO Run Server

```bash
npm start
```

- For Swagger documentation

```bash
http://localhost:3000/api-docs
```
